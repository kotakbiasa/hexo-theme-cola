Cara Menggunakan hexo-theme-cola

hexo-theme-cola adalah tema blog berbasis Hexo yang bergaya sederhana.

Tidak ada antarmuka yang mencolok, tidak ada fitur yang rumit. Tujuannya adalah agar pengguna dapat lebih fokus menulis blog.

Mulai

Langkah pertama adalah menginisialisasi proyek dengan Hexo:

hexo init blog

Setelah cloning selesai, masuk ke proyek dan clone tema ke dalam folder themes:

git clone git@github.com:Aizener/hexo-theme-cola.git

Setelah langkah-langkah di atas selesai, Anda dapat mulai mengkonfigurasi file _config.yml di direktori root. Ada dua tempat yang perlu dikonfigurasi:

  • Nonaktifkan opsi highlight dan aktifkan opsi prismjs:

    highlight:
      enable: false    # ubah true menjadi false
      line_number: true
      auto_detect: false
      tab_replace: ''
      wrap: true
      hljs: false
    prismjs:
      enable: true    # ubah false menjadi true
      preprocess: true
      line_number: true
      tab_replace: ''

  • Ubah konfigurasi theme menjadi: tema hexo-theme-cola:

    theme: hexo-theme-cola

Setelah langkah-langkah di atas selesai, lanjutkan dengan menginstal dependensi yang diperlukan.

Setelah instalasi selesai, jalankan perintah berikut untuk menghasilkan ulang tema dan memulai layanan.

hexo clean & hexo g & hexo s

Membuat Halaman Baru

Setelah tema terinstal, Anda perlu membuat beberapa halaman baru dengan menjalankan perintah berikut:

  1. Buat halaman log: hexo new page log

  2. Buat halaman tautan: hexo new page link

  3. Buat halaman tentang: hexo new page about

  4. Buat halaman kategori: hexo new page categories

  5. Buat halaman tag: hexo new page tags

  6. Buat halaman alat: hexo new page tools

Setelah halaman-halaman tersebut dibuat, Anda dapat mengkonfigurasi informasi yang sesuai di file konfigurasi _config.yml tema. (Buat halaman sesuai kebutuhan)

Halaman log perlu dibuat dengan perintah berikut:

hexo new page --path log/logName "logName"

Perintah ini akan membuat file dengan title logName di direktori log (ubah menjadi judul yang diinginkan), kemudian tulis log seperti menulis artikel biasa.

Dengan format seperti di atas, halaman log akan ditampilkan dalam urutan dari atas ke bawah.

Modifikasi Lainnya

Anda dapat melihat bahwa ada beberapa konfigurasi di situs web yang belum dijelaskan. Mengkonfigurasi ini juga sangat mudah. Masuk ke direktori tema dan temukan file _config.yml, lalu ganti sesuai dengan konfigurasi yang sudah ada.

title: Blog Cola
    # Nama di sidebar kiri
    author_name: Cola.

    # Tag di sidebar kiri, hanya dua
    tag1: Sederhana
    tag2: Rumah

    # Tautan Github di sidebar kiri
    github_url: https://github.com/Aizener

    # Motto di sidebar kiri, dua kalimat
    motto_prev: “Bunga bisa mekar lagi, tetapi manusia tidak bisa kembali muda”
    motto_last: “Seorang anak laki-laki yang sederhana dan biasa”

    # Tanggal publikasi blog (waktu operasi)
    public_date: 2022-02-16

    # Menu di sidebar kiri
    menu:
      - [Beranda, '/', 'icon-shouye']
      - [Log Pribadi, '/log', 'icon-rizhi']
      - [Tautan Teman, '/link', 'icon-youqinglianjie']
      - ['Tentang Saya', '/about', 'icon-guanyuwomen']

    # Tautan teman: avatar, nama, tautan
    links:
     - ['https://www.baidu.com/img/flexible/logo/pc/result.png', 'Baidu.', 'http://baidu.com']

    # Pengumuman di sidebar kanan
    notice: Hello~ Semua. Saya Cola yang suka di rumah, selamat datang di blog saya, semoga Anda menemukan sesuatu yang berguna di sini!

    # Daftar musik di sidebar kanan: nama lagu, artis, tautan, sampul
